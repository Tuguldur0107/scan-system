# 🗄️ Өгөгдөл хэрхэн харах вэ?

> Таны уншсан баркодны мэдээлэл, хэрэглэгчид, төслүүд гэх мэт **БҮХ ӨГӨГДӨЛ**
> `PostgreSQL` гэдэг өгөгдлийн санд хадгалагддаг.
>
> Энэ заавар танд ӨГӨГДӨЛӨӨ ӨӨРӨӨ ХАРАХ 4 ӨӨР АРГА-ыг тайлбарлана.

---

## 📋 Эхлээд мэдэх зүйлс

**Холболтын мэдээлэл** (`server/.env` файлаас):

| Талбар | Утга |
|---|---|
| Host (сервер) | `localhost` |
| Port (порт) | `5432` |
| Database (сангийн нэр) | `scan_system` |
| User (хэрэглэгч) | `scan_user` |
| Password (нууц үг) | `scan_pass` |

Эдгээрийг доорх бүх аргад ашиглана.

---

## ⚡ Сервер асаалттай байх ёстой!

Өгөгдөл харахаасаа өмнө өгөгдлийн сан **ажиллаж байх ёстой**:

```powershell
cd scan-system/server
docker-compose up -d
```

*(`-d` гэдэг нь арын талд асаана гэсэн үг)*

Шалгах:
```powershell
docker ps
```
`postgres` гэсэн мөр харагдвал амжилттай.

## 🎯 Арга 1: DBeaver (ЗӨВЛӨГДСӨН — ХАМГИЙН ХЯЛБАР) ⭐

**DBeaver** бол өгөгдлийн сан харах **үнэгүй** программ. Excel шиг харагддаг.

### Суулгах
1. Линк рүү ор: https://dbeaver.io/download/
2. **Community Edition**-ыг татаж авах
3. Суулгах (дараагийн товчлууруудыг дарсаар л)

### Холболт үүсгэх
1. DBeaver-ийг нээх
2. Зүүн дээд буланд **"New Database Connection"** (залгуур бүхий нэмэх тэмдэг) товчийг дарах
3. **PostgreSQL**-ийг сонгох → **Next**
4. Дараах талбаруудыг бөглөнө:

   ```
   Host:     localhost
   Port:     5432
   Database: scan_system
   Username: scan_user
   Password: scan_pass
   ```

5. **Test Connection** товчийг дарж шалгах (ногоон тик гарвал ОК)
6. **Finish** товчийг дарах

### Өгөгдөл харах
1. Зүүн талд `scan_system` → **Schemas** → `public` → **Tables** гэж нээх
2. `scans` хүснэгт дээр **давхар даралт**
3. Дээрх табаас **"Data"** сонгох
4. **Excel шиг бүх баркод харагдана!** 🎉

### Ашигтай зүйлс
- 🔍 Багана дээр дарж **эрэмбэлэх**
- 🎯 Дээд талын шүүлтүүрээр **хайх**
- 📊 **"Export Data"** товчоор Excel/CSV руу татаж авах
- 📝 Шууд хүснэгт дотор засах боломжтой

## 💻 Арга 3: Docker-ийн дотор `psql` ашиглах

Энэ нь **нэмэлт программ суулгахгүйгээр** шууд харах арга. Гэхдээ бичгийн командуудаар ажиллана.

### Алхам 1: Өгөгдлийн сан руу холбогдох

```powershell
docker exec -it scan-system-postgres-1 psql -U scan_user -d scan_system
```

> ⚠️ Container-ийн нэр өөр байж болно. `docker ps` командаар шалгаарай.

### Алхам 2: Үндсэн командууд

Холбогдсоны дараа `scan_system=#` гэсэн тэмдэг гарна. Дараах командуудыг бичнэ:

```sql
-- Бүх хүснэгтийг харах
\dt

-- Бүх скан харах
SELECT * FROM scans;

-- Сүүлийн 10 скан
SELECT barcode_value, scanned_at, notes
FROM scans
ORDER BY scanned_at DESC
LIMIT 10;

-- Хэдэн скан байгаа?
SELECT COUNT(*) FROM scans;

-- Хэрэглэгчдийн жагсаалт
SELECT username, role FROM users;

-- Төслүүд
SELECT name, is_open FROM projects;

-- Өнөөдрийн скан
SELECT * FROM scans
WHERE scanned_at::date = CURRENT_DATE;

-- Гарах
\q
```

## 🌐 Арга 4: pgAdmin (Вэб хөтөчөөр)

**pgAdmin** бол өгөгдлийн сангийн албан ёсны хэрэгсэл. DBeaver-ийн адилтгаа, гэхдээ вэб хөтөчөөр ажилладаг.

### Docker-оор асаах (ХЯЛБАР)

`scan-system/server/docker-compose.yml` файл дотор дараах хэсгийг нэмж болно:

```yaml
  pgadmin:
    image: dpage/pgadmin4
    environment:
      PGADMIN_DEFAULT_EMAIL: admin@admin.com
      PGADMIN_DEFAULT_PASSWORD: admin
    ports:
      - "5050:80"
    depends_on:
      - postgres
```

Дараа нь:
```powershell
docker-compose up -d
```

Хөтөч дээр очих: http://localhost:5050
- Имэйл: `admin@admin.com`
- Нууц үг: `admin`

> 💡 Хүсвэл би энэ өөрчлөлтийг хийж өгөх болно — зүгээр хэлээрэй.

## 📊 Нийтлэг асуулт-хариулт

### "Өнөөдөр хэд скан хийсэн бэ?"
```sql
SELECT COUNT(*) FROM scans
WHERE scanned_at::date = CURRENT_DATE;
```

### "Хэн хамгийн их скан хийсэн бэ?"
```sql
SELECT u.username, COUNT(s.id) AS scan_count
FROM scans s
JOIN users u ON s.user_id = u.id
GROUP BY u.username
ORDER BY scan_count DESC;
```

### "Тодорхой баркод хэзээ уншигдсан бэ?"
```sql
SELECT scanned_at, notes
FROM scans
WHERE barcode_value = 'УНШСАН_БАРКОДЫН_ДУГААР';
```

### "Тухайн төслийн бүх скан"
```sql
SELECT s.barcode_value, s.scanned_at, u.username
FROM scans s
JOIN users u ON s.user_id = u.id
JOIN projects p ON s.project_id = p.id
WHERE p.name = 'ТӨСЛИЙН_НЭР'
ORDER BY s.scanned_at DESC;
```

### "Давхардсан баркод байна уу?"
```sql
SELECT barcode_value, COUNT(*) AS count
FROM scans
GROUP BY barcode_value
HAVING COUNT(*) > 1;
```
