# Scan System — Бизнес хэрэгцээний баримт бичиг

## 1. Төслийн тойм

**Scan System** нь олон байгууллагын (multi-tenant) баркод сканнерийн платформ бөгөөд агуулахын бүртгэл, хээрийн аудит, болон даалгаварт суурилсан сканнердах ажлын урсгалд зориулагдсан. Систем нь мобайл болон вэб дээр ажиллах боломжтой, офлайн горимд скан хийж, дараа нь серверт синхрончлох чадвартай.

---

## 2. Бизнес зорилго

| # | Зорилго | Тайлбар |
|---|---------|---------|
| 1 | Баркод сканнердах үйл ажиллагааг цахимжуулах | Цаасан дээрх бүртгэлийг бүрэн орлуулж, алдааг бууруулах |
| 2 | Олон байгууллагын тусгаарлагдсан орчин | Байгууллага бүр өөрийн хэрэглэгч, төсөл, дата-тай ажиллах |
| 3 | Офлайн ажиллагааг дэмжих | Сүлжээгүй орчинд скан хийж, дараа нь batch-ээр синхрончлох |
| 4 | Хяналт, тайлан | Удирдлага дашбоардаар дамжуулан бодит цагийн мэдээлэл авах |
| 5 | Өгөгдлийн экспорт | CSV форматаар дата татаж, гадаад системтэй холбогдох |

## 3. Хэрэглэгчийн дүрүүд (Roles)

### 3.1 Супер Админ (`super_admin`)
- **Хэн**: Системийн ерөнхий администратор
- **Хариуцлага**: Бүх байгууллагуудыг удирдах
- **Эрх**:
  - Шинэ байгууллага (tenant) үүсгэх, засварлах, устгах
  - Байгууллага бүрийн хэрэглэгчдийг удирдах
  - Бүх байгууллагын дашбоард, дата харах
  - Байгууллагын тохиргоо өөрчлөх

### 3.2 Байгууллагын Админ (`tenant_admin`)
- **Хэн**: Байгууллагын дотоод удирдлага
- **Хариуцлага**: Өөрийн байгууллагын үйл ажиллагааг удирдах
- **Эрх**:
  - Төсөл (даалгавар) үүсгэх, нээх/хаах
  - Байгууллагын хэрэглэгчдийг удирдах (нэмэх, идэвхжүүлэх/идэвхгүйжүүлэх)
  - Дашбоард харах (нийт скан, хэрэглэгчээр, төслөөр)
  - Дата экспорт хийх (CSV)

### 3.3 Оператор (`operator`)
- **Хэн**: Скан хийх ажилтан
- **Хариуцлага**: Баркод сканнердах, дата илгээх
- **Эрх**:
  - Баркод скан хийх
  - Хүлээгдэж буй сканнууд харах, засварлах, устгах
  - Түүх (history) харах
  - Batch синхрончлол хийх

## 4. Функциональ шаардлагууд

### 4.1 Нэвтрэх систем (Authentication)

| # | Шаардлага | Тайлбар |
|---|-----------|---------|
| AUTH-01 | Нэвтрэх | Байгууллагын код (slug), хэрэглэгчийн нэр, нууц үгээр нэвтрэх |
| AUTH-02 | Токен менежмент | JWT access token (15 мин) + refresh token (30 хоног) |
| AUTH-03 | Токен сэргээх | Refresh token-оор шинэ access token авах |
| AUTH-04 | Нууц үг солих | Хэрэглэгч өөрийн нууц үгийг солих боломж |
| AUTH-05 | Гарах | Refresh token хүчингүй болгох |
| AUTH-06 | Сессия сэргээх | Апп дахин нээхэд автомат нэвтрэх (secure storage) |
| AUTH-07 | Демо горим | Туршилтын нэвтрэх мэдээлэл (хөгжүүлэлтийн орчинд) |

### 4.2 Баркод сканнер

| # | Шаардлага | Тайлбар |
|---|-----------|---------|
| SCAN-01 | Камераар скан | Мобайл камераар баркод уншуулах (бодит цаг) |
| SCAN-02 | Гар оруулга | Баркод утгыг гараар оруулах боломж (вэб, гараар) |
| SCAN-03 | Давхардал хамгаалалт | Ижил баркодыг 1.2 секундын дотор давтан скан хийхээс сэргийлэх |
| SCAN-04 | Дууны мэдэгдэл | Скан хийхэд дуут болон чичиргээт (haptic) мэдэгдэл |
| SCAN-05 | Гэрэл (torch) | Камерын гэрэл асаах/унтраах |
| SCAN-06 | Төсөл сонголт | Скан хийхдээ аль төсөлд хадгалахаа сонгох |
| SCAN-07 | Визуал санал хүсэлт | Уншсан баркодын утгыг дэлгэцэнд харуулах |

### 4.3 Офлайн дараалал ба синхрончлол

| # | Шаардлага | Тайлбар |
|---|-----------|---------|
| SYNC-01 | Локал хадгалалт | Скан хийсэн дата-г утасны санах ойд хадгалах |
| SYNC-02 | Хүлээгдэж буй дараалал | Синхрончлогдоогүй сканнуудыг жагсаалтаар харах |
| SYNC-03 | Batch синхрончлол | Бүх хүлээгдэж буй сканнуудыг нэг удаа илгээх (500 хүртэл) |
| SYNC-04 | Batch ID (send_id) | Илгээлт бүрт `SEND-YYYYMMDD-HHMMSS-XXXX` форматын ID үүсгэх |
| SYNC-05 | Алдааны удирдлага | Синхрончлол амжилтгүй бол алдааг харуулж, дахин оролдох боломж |
| SYNC-06 | Скан засварлах/устгах | Илгээгээгүй сканнуудыг засварлах, устгах боломж |

## 7. Ажлын урсгал (Workflows)

### 7.1 Оператор — Скан хийх урсгал

```
Нэвтрэх → Төсөл сонгох → Баркод скан хийх → Локал дараалалд хадгалах
→ Хүлээгдэж буй жагсаалт шалгах → "Бүгдийг синхрончлох" дарах
→ Серверт batch илгээх → Түүхэнд шилжих
```

### 7.2 Админ — Өдөр тутмын хяналт

```
Нэвтрэх → Дашбоард харах → Хэрэглэгчээр/Төслөөр статистик шалгах
→ Шаардлагатай бол шинэ төсөл үүсгэх → Дата экспорт хийх (CSV)
```

### 7.3 Супер Админ — Байгууллага удирдах

```
Нэвтрэх → Байгууллагын жагсаалт → Шинэ байгууллага үүсгэх
→ Админ хэрэглэгч тохируулах → Байгууллагын дашбоард шалгах
```
