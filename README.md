# 📱 Scan System — Баркод Сканнердах Программ

> Кодын мэдлэггүй хүнд зориулсан ерөнхий гарын авлага.
> Бүх тайлбар **монгол хэл дээр**.

---

## 🎯 Энэ программ юу хийдэг вэ?

Ажилчид **гарын утсаараа баркод унших** → мэдээлэл нь автоматаар **оффисын сервер рүү** илгээгдэнэ → Админ хүн бүх уншилтыг нэг дороос хардаг.

**Интернэтгүй үед** ч скан хийж болно — утсан дээр түр хадгалаад дараа нь илгээгддэг.

---

## ⚡ 1 Хуудас Cheat Sheet

Өдөр бүр доорхыг л хийнэ:

### 1) Бүгдийг нэг товчоор асаах (ЗӨВЛӨДӨГ)

```powershell
cd scan-system/scripts
.\start_all.cmd web
```

Төрлүүд:
- `.\start_all.cmd web` → сервер + вэб
- `.\start_all.cmd android` → сервер + android
- `.\start_all.cmd both` → сервер + android + вэб

### 2) Нэвтрэх

- Tenant: `system`
- Username: `admin`
- Password: `admin123`

### 3) Скан илгээх

1. `Pending` руу орно
2. `Бүгдийг илгээх` дарна
3. Амжилттай бол `History`-д орно

### 4) Илгээгдсэн баркодуудаа харах

- Апп дээр: `History`
- Вэб дээр: `Data`
- SQL дээр: `scans` хүснэгт

### 5) Хаах

Нээгдсэн терминал бүр дээр `Ctrl + C`.

## 💾 ӨГӨГДӨЛ ХААНА ХАДГАЛАГДДАГ ВЭ? ⭐

**2 газар** хадгалагддаг:

### 📱 1) Утсан дээр (түр зуур)
- **Хаана:** `app/lib/data/local/local_scan.dart`
- **Яагаад:** Интернэтгүй үед ч ажиллахын тулд
- **Төлөв:** Илгээгдсэний дараа автоматаар устдаг

### 🗄️ 2) Серверийн өгөгдлийн санд (**үндсэн хадгалалт**)
- **Ямар сан:** PostgreSQL (дэлхий нийтийн стандарт)
- **Хаана ажилладаг:** Docker дотор
- **Холбогдох хаяг:** `server/.env` файлд бичигдсэн
  ```
  postgres://scan_user:scan_pass@localhost:5432/scan_system
  ```
- **Хүснэгтүүд:** `server/lib/db.dart` дотор тодорхойлогдсон

### Өгөгдлийн сангийн 7 хүснэгт

| Хүснэгт | Юу хадгалдаг |
|---|---|
| `tenants` | 🏢 Компаниуд (олон компани нэг программ ашиглах боломжтой) |
| `users` | 👥 Хэрэглэгчид (superadmin / tenant_admin / operator) |
| `projects` | 📋 Төслүүд (жш: "Уулын уурхай #1") |
| **`scans`** ⭐ | **БҮХ УНШСАН БАРКОД** |
| `refresh_tokens` | 🎫 Нэвтрэх түлхүүрүүд |
| `audit_log` | 📝 Хэн юу хийсэн бүртгэл |
| `_migrations` | 🔄 Сангийн шинэчлэлт бүртгэл |

### `scans` хүснэгт — таны ҮНДСЭН өгөгдөл

| Багана | Юу |
|---|---|
| `id` | Өвөрмөц дугаар |
| `tenant_id` | Аль компанийнх |
| `project_id` | Аль төслийнх |
| `user_id` | Хэн уншсан |
| **`barcode_value`** ⭐ | **БАРКОДНЫ ДУГААР** |
| `barcode_format` | Ямар төрлийн баркод (QR, Code128 г.м.) |
| `scanned_at` | Хэзээ уншсан |
| `synced_at` | Хэзээ сервер рүү илгээгдсэн |
| `notes` | Тэмдэглэл |
| `metadata` | Нэмэлт мэдээлэл (JSON) |

## ▶️ Яаж асаах вэ?

### Алхам 1: Серверийг асаах ⭐ (ШИНЭ — НЭГ ТОВЧЛУУР)

```powershell
cd scan-system/scripts
.\start_server.cmd
```

Энэ автоматаар дараах зүйлсийг хийнэ:
1. Docker дотор өгөгдлийн санг асаана
2. Компьютерийн LAN IP-г олно (утсаар ашиглах)
3. dart_frog серверийг асаана
4. Утсанд холбогдох URL-ыг дэлгэцэд харуулна

### Алхам 2: Программыг асаах

**Вэб хувилбар (хамгийн хялбар):**
```powershell
cd scan-system/scripts
.\run_web.cmd
```

**Android утас:**
```powershell
cd scan-system/scripts
.\run_android.cmd
```

*(Сервер ажиллаж байгаа үед `run_android.cmd` автоматаар зөв IP-г олно.)*
